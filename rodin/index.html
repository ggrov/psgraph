This webpage aims to present resources used for our tool presentation. To use those resources, first download Plugin with associated files from the following links

For Linux users, please download Tinker Core for Rodin from here
For Windows users, please download Tinker Core for Rodin from here

    Once you have extracted the files

    • 1. Simply extract the jar file to the "plugins" folder under Rodin installation folder. Lauch Rodin
    • 2. Once Rodin is launched, go to menu -> window -> preference -> Tinker preference page 2.1 Leave the Default Directory empty 2.2 Browse to set the

    Then open the following demo file in the pp to see if the tool can run correctly

    $SOME_PATH/tinker/psgraph/src/core/demo/TACAS16/

    The line

    PPIntf.set_psg_goal

    is to ask pp to connect gui for proofs. Note that to enable gui to accept the connection from pp, the following need to be clicked , once the connection is ready the icon will become . Details on how to start the GUI can be found in Tinker's user guide.
